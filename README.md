# МЕОТИДА · Маркировочный стол

Одностраничное приложение (один файл `index.html`, работает офлайн — откройте в
браузере). Объединяет четыре рабочих инструмента для кодов «Честный знак» и ЭДО
в одном месте, в общем оформлении (тёмная / светлая тема).

## Вкладки

1. **Недостача** — вычисляет недостачу пачек и блоков в коробе с таможни:
   коды из письма склада → «Инфо по марке» → «Состав марки» → «История
   агрегации» (2 этапа) → автоматический расчёт недостачи, недостающие пачки
   в разорванных блоках, **яркое выделение чужих блоков и пачек из другого
   короба**, статусы/владельцы отсутствующих блоков и готовое письмо
   (кнопка «Написать в Thunderbird» — .eml). Стандартные имена файлов
   выгрузок с кнопкой «Скопировать имя». Сессии (до 10) с поиском по коду.
2. **План / Факт** — сравнение двух списков кодов (чего нет в факте и что
   лишнее).
3. **GTIN / МЦ** — локальная база минимальных цен (mcGtinCop) по GTIN:
   поиск, пополнение из JSON «Честного знака», «Скачать HTML с базой»
   (автономный файл со всей базой внутри) и анализ ошибок ГИС МТ по УПД —
   где цена в документе ниже минимальной.
4. **УКД → иУКД** — конвертер корректировочных документов ЭДО: новый UUID v4
   в имени и в `ИдФайл`, вставка строки `ИспрКСчФ`; во второй вкладке —
   правка цен, сумм, реквизитов и кодов маркировки. Один файл — XML,
   несколько — ZIP.
5. **СвВыбытияМАРК** — вывод кодов из оборота для собственных нужд: добавляет
   `<ТекстИнф Идентиф="СвВыбытияМАРК" Значен="3"/>` в блок `ИнфПолФХЖ1`,
   проставляет новый UUID v4 (имя файла + `ИдФайл`) и, при необходимости,
   `ДатаСчФ`. Скачивает готовый УПД (один — XML, несколько — ZIP) и открывает
   валидацию Диадок.
6. **Как пользоваться** — инструкция по всем инструментам с картинками.

## Ещё в этой версии

- В «Недостаче» — блок **«Загрузить все файлы разом»**: несколько выгрузок
  или архив `.zip` раскладываются по шагам автоматически (по имени файла или
  содержимому), плюс единый список имён всех файлов с копированием.
- В «План / Факт» — кнопка **«＋ Добавить ещё сравнение»**: несколько
  независимых сравнений на одном экране (например, коды блоков и коды пачек).
- В подвале — версия, дата и «Разработано Викторией Саламашко».

## Общие возможности

- **Тема** — переключатель светлая / тёмная в правом верхнем углу
  (запоминается).
- **Загрузка файлов** — почти у каждого поля кнопка «↑ файл»: можно вставить
  текст или загрузить **.txt**, **.xls**, **.xlsx** (таблица из Excel
  читается автоматически, кириллица в старых `.xls` без кодовой страницы
  декодируется как cp1251).
- **Очистка** — у каждого поля кнопка «✕ очистить», а в шапке «✕ Очистить
  всё» очищает поля во всех вкладках (сохранённые сессии и база GTIN
  остаются).

## Технически

- Один самодостаточный HTML-файл. Внешняя зависимость только одна — Google
  Fonts (Unbounded / Manrope / JetBrains Mono); без интернета шрифты
  заменяются системными, всё продолжает работать.
- Чтение `.xls`/`.xlsx` — встроенный SheetJS; изображения инструкции —
  data-URI.
- Дизайн — по `design_system_konspekty_v2.md` («Dark Editorial»): тёмный фон,
  волосяные линии, дисплейная типографика, пыльно-голубой + шампань акценты,
  полноценная светлая тема.
- Всё работает локально в браузере — данные никуда не отправляются.
